sample.estimate {MSTAT}R Documentation

Sample size for estimation

Description

This command gives minimum sample size for estimation of mean and proportion

Usage

sample.estimate(estimate, sig.level = 0.05)

Arguments

estimate

The quantity you want to estimate either "mu" or "p"

sig.level

Desierd significance level default is 0.05

Value

gives output n: minimum sample size
